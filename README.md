## Features 🍔

- React + Next JS
- Tailwind CSS
- Power Packed with React and Next Modules
- Next JS API Router for fetching Kavithai data
- PWA Support
- Vercel Cache

## Installation 📦

- Clone this repo or Download

```sh
git clone https://github.com/mskian/next-tamil-sms
cd next-tamil-sms
yarn
```

- Test the site

```sh
yarn dev
```

- Build the site

```sh
yarn build
```

- Test the production Build on Localhost

```sh
yarn start
```
